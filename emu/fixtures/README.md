# Independent DNG fixtures

These fixtures are generated by TinyDNG rather than Banksia's DNG writer, so
the native decoder is not only round-tripped against its sibling encoder.

- Upstream: `https://github.com/syoyo/tinydng`
- Revision: `5e5686e468738ada1b4edc5b334c1fe548f91a8d`
- Upstream licence: MIT (`Copyright (c) 2016-Present Syoyo Fujita and contributors`)
- Fixture pixel data: deterministic synthetic values authored for Banksia;
  released under Banksia's project licence.
- `tinydng-big-endian.dng`: big-endian, uncompressed, 8×6 RGGB, 16-bit.
- `tinydng-lossless-jpeg.dng`: little-endian, Compression=7 lossless JPEG,
  16×3 RGGB, 16-bit. TinyDNG's encoder folds the source's paired rows into
  one scanline, hence the stored dimensions.

Generation:

```sh
git clone https://github.com/syoyo/tinydng /tmp/tinydng
git -C /tmp/tinydng checkout 5e5686e468738ada1b4edc5b334c1fe548f91a8d
c++ -std=c++11 -O2 -I/tmp/tinydng tinydng-fixture.cc -o /tmp/tinydng-fixture
/tmp/tinydng-fixture tinydng-big-endian.dng tinydng-lossless-jpeg.dng
```
